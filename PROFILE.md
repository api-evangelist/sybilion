# Sybilion

Sybilion is a decision layer for industrial companies, providing economic and causal forecasting that connects external market dynamics to internal exposure for procurement, trading and risk teams. It exposes forecasts, their drivers, and alerts via a REST API and a hosted MCP server.

- **Provider:** https://sybilion.dev/
- **Documentation:** https://sybilion.dev/docs/
- **Base URL:** https://api.sybilion.dev
- **Tags:** Industrial market intelligence, Commodity price forecasting, Economic forecasting, Time-series forecasting, Procurement, Supply-chain risk, Trading analytics, AI decision support, MCP
- **Public API:** yes
- **Agent-native (MCP / llms.txt / skills):** yes
- **Confidence:** 90/100

## APIs

- **Sybilion Operational API** — REST API exposing asynchronous forecasts with quantile bands and driver attribution, ranked external drivers, alerts, account/usage, tiers, API keys, and billing. Uses HTTP Bearer authentication with API keys (sk_ops_...) or Auth0 JWT session tokens. (`https://api.sybilion.dev`)
- **Sybilion MCP Server** — Hosted MCP server providing agent-native tools for forecasts, drivers, alerts, regions/categories and account/usage, with OAuth browser approval flow. Documented integrations include ChatGPT, Claude, and the TradingView Remix extension. (`https://mcp.sybilion.dev/mcp`)
